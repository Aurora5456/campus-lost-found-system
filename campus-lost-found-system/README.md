# 校园失物招领系统

这是一个基于 Python Flask 的期末软件大作业项目，使用 MySQL 保存学生、管理员、帖子和私信数据。项目包含学生端和管理员端，支持发帖、搜索、图片上传、权限控制和站内私信。

## 技术栈

- Python Flask
- Flask-SQLAlchemy
- MySQL + PyMySQL
- HTML / CSS / JavaScript / Jinja2
- Werkzeug 密码哈希

## 安装依赖

```bash
pip install -r requirements.txt
```

## 配置 MySQL

默认配置在 `config.py` 中：

```text
数据库主机：localhost
数据库端口：3306
数据库名称：lost_found_system
数据库用户：root
数据库密码：123456
```

如果你的 MySQL 用户名或密码不是 `root / 123456`，请修改 `config.py` 中的 `DB_USER`、`DB_PASSWORD`，或设置环境变量 `DB_USER`、`DB_PASSWORD`、`DB_HOST`、`DB_PORT`、`DB_NAME`。

## 创建数据库

先登录 MySQL，然后执行：

```sql
CREATE DATABASE lost_found_system DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

也可以执行项目中的 `database.sql` 创建数据库和表结构。

## 初始化数据库

推荐双击或运行：

```bash
init_db.bat
```

也可以手动运行：

```bash
python init_db.py
```

该脚本会自动创建 `lost_found_system` 数据库，清空并重建数据表，然后插入测试学生账号、管理员账号和示例帖子。

如果提示 MySQL 连接失败，请检查：

- MySQL 服务是否已启动
- 是否已经创建 `lost_found_system` 数据库
- `config.py` 中的账号、密码和端口是否正确

## 启动项目

推荐使用一键启动脚本，它会后台启动 Flask、写入日志、检测端口并打开浏览器：

```bash
start.bat
```

停止服务：

```bash
stop.bat
```

手动启动方式：

```bash
python app.py
```

浏览器访问：

```text
http://127.0.0.1:5000
```

## 主要功能完成情况

- 学生端登录、退出
- 学生查看失物招领帖子列表
- 按标题、物品名称、描述、地点模糊搜索
- 学生发布帖子
- 图片上传到 `static/uploads`
- 学生只能编辑和删除自己的帖子
- 帖子详情页展示发布人和物品信息
- 帖子详情页联系发布人
- 学生私信列表
- 聊天详情页发送和查看消息
- 管理员登录后台
- 管理员查看、搜索、编辑、删除所有帖子
- 删除帖子时同步清理关联私信
- 密码使用 Werkzeug 哈希保存
- 后端路由进行权限校验

## 答辩演示流程

1. 打开 `http://127.0.0.1:5000`，使用学生账号 `20230001 / 123456` 登录。
2. 在首页查看示例失物招领帖子，演示卡片式列表和搜索框。
3. 点击“发布帖子”，填写标题、物品名称、类型、描述、地点、联系说明，并上传一张 `jpg/png/gif` 图片。
4. 发布成功后进入帖子详情页，展示图片、物品信息、发布人学号和发布时间。
5. 进入“我的帖子”，编辑刚才发布的帖子，说明学生只能修改自己的帖子。
6. 回到首页，用关键词按标题、物品名称、描述或地点进行模糊搜索。
7. 退出登录，使用另一个学生账号 `20230002 / 123456` 登录。
8. 打开刚才的帖子详情页，点击“联系发布人”，进入私聊页面并发送文字消息。
9. 进入“我的私信”，展示私信列表和聊天记录。
10. 退出学生账号，进入管理员登录页，使用 `admin / admin123` 登录后台。
11. 在管理员后台搜索帖子，查看发布人信息。
12. 管理员编辑或删除学生帖子，说明管理员可管理全部帖子。
13. 打开 MySQL 客户端，查看 `students`、`admins`、`posts`、`messages` 表和数据记录。
